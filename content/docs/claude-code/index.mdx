---
title: Claude Code Plugin
description: Use the Nova plugin to build CommCare apps from inside Claude Code.
---

The Nova plugin connects Claude Code to your CommCare Nova account. Once it's installed and signed in, you can ask Claude to build a new CommCare app, modify an existing one, or push an app to CommCare HQ. Claude does the work using your Claude Code plan; Nova doesn't bill anything on top.

- [Quick start](./quickstart.mdx) — install the plugin, sign in, and build your first app.
- [Commands](./commands.mdx) — reference for every `/nova:*` command.
- [Manage access](./access.mdx) — disconnect Claude Code from Nova or uninstall the plugin.
